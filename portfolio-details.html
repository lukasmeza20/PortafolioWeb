Información Proyecto personal

  • Categoria: Tienda Virtual

Este proyecto de tienda virtual fue construido con Django en el backend que me entrego la autenticación en JWT además de otras funcionalidades como el carrito de compras y la programación de los objetos a travez del panel de administración , por otro lado el frontend del sistema fue hecho con react js que me permite al usuario tener una mejor experencia al utilizar la tienda con sus componentes y hooks permitiendo que el sistema sea rapido y sin tiempo de carga, además de usar redux para gestionar el estado del sitio.
